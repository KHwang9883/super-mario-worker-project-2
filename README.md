# SMWP 2

<p align="center"><img src="icon.png" alt="SMWP 2 Logo"></p>

🔨 **SMWP 2 (Super Mario Worker Project 2)** is the successor to Super Mario Worker Project, developed by INNOVATION LEAP. While maintaining compatibility with SMWP 1 levels, it brings greater extensibility to SMWP.

- English | [中文](README_zh-cn.md)

## Features

- 🚀 **New Engine:** Game Engine migrated to Godot Engine 4, bringing more features and better performance.
- 🔗 **SMWP 1 Compatibility:** Keep the same level format and game mechanics as SMWP 1, with extensions built on top.
- 👐 **Open Source**：SMWP 2 will be open-source, allowing players and level creators to freely modify and customize.

## Download

SMWP 2 is currently under development. The current goal is to implement SMWP 1 level functionality. The level editor is not yet available.

**[Latest ci build available here.](https://github.com/InnovationLeap/super-mario-worker-project-2/actions/workflows/godot-ci.yml)** They're not yet stable, use at your own risk.
