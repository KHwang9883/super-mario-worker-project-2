# SMWP 2

<p align="center"><img src="icon.png" alt="SMWP 2 Logo"></p>

🔨 **SMWP 2 (Super Mario Worker Project 2)** 是由 INNOVATION LEAP 开发的全新 Mario Worker，在尽可能兼容 SMWP 1 关卡体验的基础上，为 SMWP 带来更多的可扩展性。

## 特性

- 🚀 **新引擎**：全面转向 Godot Engine 4，带来更多的功能和性能优化。
- 🔗 **兼容 SMWP 1 生态**：保持与 SMWP 1 相同的关卡格式和游戏机制，并基于其进行扩展。
- 👐 **开源**：SMWP 2 将全面开源，允许玩家和关卡创作者自由修改定制。

## 下载

SMWP 2 仍处于开发阶段，尚未正式发布。当前开发版本可运行 SMWP 1 的关卡作品。

**[最新构建可点此下载。](https://github.com/InnovationLeap/super-mario-worker-project-2/actions/workflows/godot-ci.yml)**
